⚙️ Step 2 — News Collector

A Python service that collects and stores news.

Example:

import feedparser

rss_url = "https://www.reuters.com/markets/rss"

feed = feedparser.parse(rss_url)

articles = []

for entry in feed.entries:
    articles.append({
        "title": entry.title,
        "summary": entry.summary,
        "link": entry.link,
        "published": entry.published
    })

print(articles[:3])

Store them in PostgreSQL.

🗄 Database Table
CREATE TABLE news_articles (
    id SERIAL PRIMARY KEY,
    title TEXT,
    summary TEXT,
    content TEXT,
    source VARCHAR(50),
    published_at TIMESTAMP,
    embedding VECTOR(384),
    sentiment_score FLOAT,
    impact_score FLOAT
);

Use pgvector.

🧹 Step 3 — News Cleaning

Remove noise.

Example:

def clean_news(text):

    text = text.replace("\n"," ")
    text = text.strip()

    return text

Combine title + summary:

article_text = title + ". " + summary
🧠 Step 4 — Embedding Model

Convert articles into vectors.

Best free model:

sentence-transformers/all-MiniLM-L6-v2

from sentence_transformers import SentenceTransformer

model = SentenceTransformer("all-MiniLM-L6-v2")

embedding = model.encode(article_text)

Store in pgvector.

🤖 Step 5 — Sentiment Analysis LLM

Use local model or API.

Best lightweight option:

FinBERT

Financial sentiment model.

from transformers import pipeline

sentiment = pipeline(
    "sentiment-analysis",
    model="ProsusAI/finbert"
)

result = sentiment(article_text)

Output:

[
 {
  "label": "positive",
  "score": 0.89
 }
]

Convert to numeric:

sentiment_map = {
 "positive": 1,
 "neutral": 0,
 "negative": -1
}
📊 Step 6 — Impact Score

Not all news is important.

Calculate impact score.

Example factors:

Example formula:

impact_score = sentiment_score * source_weight

weights = {
